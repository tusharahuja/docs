Chemaxon is a leader in the field of Cheminformatics, providing chemical software development tools to the Pharmaceutical and Biotechnology industries. Chemaxonâ€™s offerings primarily consist of tools for data visualization, drawing of molecules, and searching chemical databases.

The Chemaxon JChem PostgreSQL Cartridge integrates chemical structure handling with EDB Postgres Advanced Server (EPAS) and PostgreSQL through extensions to PostgreSQL's native SQL language. It enables scientists to create, manage, and analyze chemical structures and related non-chemical data in their Postgres database. JChem users can search data by Superstructure, Substructure, Duplicate and Full fragment (exact fragment) search.
